# 📘 EPUB 图片资源压缩工具

一个用于 **批量压缩 EPUB 文件内部图片资源** 的 Windows 桌面应用。
基于 **Flutter** 开发，目标是最大化压缩图片体积，同时保持清晰度不变，并生成结构完全一致的新 EPUB 文件。

---

## ✨ 功能特性

* 选择 一个或者多个 EPUB 文件进行压缩
* 自动解析 EPUB 并提取内部图片
* 多文件独立处理，互不影响
* 每个文件保持原结构和命名规则
* 在保证清晰度前提下尽可能压缩
* 保留原图片名称 / 路径 / 格式
* 生成新的 EPUB 文件
* 支持取消压缩
* 实时显示进度与日志，全局进度 + 当前文件进度 + 当前图片进度
* 自动生成输出目录


### 📦 图片压缩规则

* 在保持清晰度前提下压缩到最小
* **不能改变图片名称**
* **不能改变图片路径**
* **不能改变图片格式**
* 支持 JPEG / PNG / WebP 等常见格式

### 📚 新 EPUB 生成规则

* **EPUB 文件名不变**
* **EPUB 内部文件路径 / 名称 / 格式不变**
* 仅替换压缩后的图片资源
* 输出到用户指定目录或自动生成目录

---

## 🖼️ UI 界面说明

### 主界面

* 简洁直观的桌面应用界面
* 支持明亮与暗黑主题（可选）
- 压缩参数配置
  - JPEG质量设置
  - 是否跳过小图设置
  - PNG优化设置
  - 输出目录设置
* 支持多选或逐个添加文件
* 可选输出路径（不选则自动生成）
* 可选压缩参数（质量、是否跳过小图）
* 实时进度条、当前处理文件显示
* 滚动日志窗口展示压缩详情
* 支持一键取消压缩
* 输出目录可统一设置或单独生成

## 🚀 使用方法

### 1. 下载

前往 Release 页面下载对应版本的 Windows 安装包或压缩包。

### 2. 运行

直接双击运行即可，无需安装额外环境。
